# HierSAPI
### (React + TypeScript + Vite)

User interface for [HierSAPI](https://github.com/Jack-42/HierSAPI)

## Installation

```shell
npm install
```

## Run Locally

```shell
npm run dev
```

### Front-end assets

_Build:_
```shell
npx tailwindcss -i ./src/App.css -o ./dist/App.css
```
_Watch:_
```shell
npx tailwindcss -i ./src/App.css -o ./dist/App.css --watch
```
